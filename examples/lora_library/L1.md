# L1 personal-agent

## Description
Specialist LoRA for personal-agent and customer-service workflows: scheduling,
travel, booking, food delivery, retail support, banking support, telecom support,
and other practical coordination tasks.

## Routing Rules
- Choose L1 for real-world coordination involving people, time, travel, food, orders, reservations, appointments, housing, health, or customer support.
- Choose L1 when the user asks the assistant to handle a concrete personal, service, booking, order, policy, or support workflow.
- Do not choose L1 for repository repair, shell tasks, local file pipelines, or UI generation.

## Strong Signals
booking, reservation, order, delivery, appointment, travel plan, hotel, flight, train, customer service, refund, payment, schedule, family coordination, restaurant

## Positive Signals
dinner plan, hotel booking, route planning, meeting schedule, customer support, retail order, credit card, telecom, food delivery

## Negative Signals
repository, codebase, pytest, stack trace, shell command, local files, React component, dashboard layout

## Examples
- Help me book a restaurant for five people tomorrow night. => L1
- I need help changing a flight reservation under the airline policy. => L1
