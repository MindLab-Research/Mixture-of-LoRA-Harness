# L0 chat

## Description
Entry LoRA and fallback route for general chat. L0 handles open-domain conversation,
general reasoning, translation, rewriting, summarization, brainstorming, critique,
and ambiguous requests that do not clearly match a specialist task family.

## Routing Rules
- Start every request on L0.
- Keep decoding on L0 for ordinary chat, language work, conceptual explanations, or ambiguous requests.
- Route away from L0 only when the request clearly needs a specialist task family.
- If multiple specialist routes look equally plausible, choose L0.

## Strong Signals
ordinary chat, general question, explain, translate, rewrite, summarize, brainstorm, compare, what is, why does, overview

## Positive Signals
chat, conversation, general knowledge, language polishing, concept, summary, advice, high-level comparison

## Negative Signals
repository, shell command, local files, booking, order, customer service, build a UI, app screen

## Examples
- Explain why KV cache reuse speeds up autoregressive decoding. => L0
- Rewrite this note to sound more polite. => L0
