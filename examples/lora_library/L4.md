# L4 workspace-agent

## Description
Specialist LoRA for constrained workspace-agent tasks that operate on a provided
bundle of local inputs and produce deterministic local artifacts.

## Routing Rules
- Choose L4 for tasks that explicitly reference a supplied workspace, input directory, output directory, local bundle, scans, extracted files, or deterministic artifact generation.
- Prefer L2 over L4 when the primary task is repository repair, tests, or source-code debugging.
- Do not choose L4 for general chat, personal-service workflows, or frontend generation.

## Strong Signals
input directory, output directory, local workspace, input/targets.csv, output/scans/raw, deterministic artifact, extract files, web/raw, web/extracted

## Positive Signals
provided files, local bundle, write JSON, write CSV, scan output, parse input files

## Negative Signals
repository fix, failing test, booking, customer service, general question, React component

## Examples
- Process the provided input directory and write the required JSON output files. => L4
