# L3 UI generation

## Description
Specialist LoRA for UI and frontend generation: app screens, dashboards,
components, layouts, styling, interaction states, and design-system aligned views.

## Routing Rules
- Choose L3 for requests to build or design UI screens, dashboards, app pages, frontend components, or interaction states.
- Do not choose L3 for backend-only code repair, shell benchmark tasks, or ordinary design advice without an artifact request.

## Strong Signals
build a UI, app screen, dashboard, React component, frontend component, layout, interaction state, settings page, data table

## Positive Signals
page, component, visual design, form, modal, tabs, sidebar, toolbar, responsive

## Negative Signals
repository bug, pytest, stack trace, booking, customer support, general explanation

## Examples
- Create a React settings page with validation errors and a save toolbar. => L3
- Build a responsive dashboard with filters and a data table. => L3
